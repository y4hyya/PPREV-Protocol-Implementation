Privacy-Preserving Real Estate Verification

Prove property ownership, tenant eligibility, and transaction conditions using zero-knowledge proofs over TLSNotary-authenticated government records — without exposing personal data.

How It Works
Three Pillars of Trust
🔒

TLSNotary

Fetch authoritative records from government portals (e-Devlet) and produce a cryptographic proof of data authenticity — no server changes needed.

🛡

Zero-Knowledge Proofs

Evaluate predicates like "I own this property" or "income ≥ 3× rent" revealing only true/false — all PII stays private.

⛓

Smart Contracts

On-chain settlement with escrow, slashing, and state machines. Ad_Hash binding prevents bait-and-switch. Nonces prevent replay.

Use Cases
Renting & Selling

🏠 Renting

  • Ownership proof via Πown — prove you can lease
  • Income eligibility via Πelig — prove income ≥ 3× rent
  • Deposit escrowed on-chain with timeout protection
  • Key handover verified via Πtrans
  • Subletting fraud prevented by fresh ownership checks

🏢 Selling

  • Title + encumbrance verification — no active liens
  • Valuation predicate — appraised value ≥ Vmin
  • Purchase funds escrowed with extended τlock
  • Deed transfer proven via registry update proof
  • Cross-listing fraud blocked by LOCKED state
